Detailed Description

The app's main application window.

MainWindow provides access to all the functionality offered by tge application through the GUI. It also houses the main application menu.

The visual representation of a SQRL identity, represented by an instance of the IdentityModel class, is being rendered within the \MainWindow using the UiBuilder class.
